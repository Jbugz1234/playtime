## How to Use

### Step 1: Open the Dashboard
1. Open `dashboard.html` in any modern web browser (Chrome, Firefox, Edge, Safari)
2. No installation or server setup required!

### Step 2: Upload Your CSV File
**Option A: Click to Upload**
- Click the "Choose File" button
- Select your CSV file from your computer

**Option B: Drag & Drop**
- Drag your CSV file directly onto the upload area
- Release to upload

### Step 3: Explore Your Data
Once uploaded, the dashboard will automatically:
- Parse and analyze your data
- Generate visualizations
- Create predictions
- Display key statistics
- Provide AI-powered insights

### Step 4: Interact with the Co-Pilot
1. Click the robot icon (bottom-right corner)
2. Type your questions or commands
3. Get instant AI-powered responses
4. Request dashboard updates

## Troubleshooting

**Dashboard doesn't load?**
- Ensure you're using a modern browser
- Check browser console for errors
- Try disabling browser extensions

**CSV not parsing?**
- Verify file is in CSV format
- Check for proper comma separation
- Ensure headers are present

**Charts not displaying?**
- Check internet connection (for CDN libraries)
- Verify CSV has numeric data
- Try a different browser

**Co-Pilot not responding?**
- Upload a CSV file first
- Check for clear, specific questions
- Try the "help" command
